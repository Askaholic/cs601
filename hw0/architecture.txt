Registers:
  There are 8 registers (0 - 7)

Instructions:
  Each instruction is 16-bits wide

  0x0mmj - LOAD from mem location m into register j
  0x1mmj - STORE from register j into mem location m
  0x2#jk - ADD value in register k to register j. Bits in # are ignored.
  0x3iij - ADD IMMEDIATE. Add value ii to register j.
  0x4iij - BRANCH IF EQUAL TO ZERO.
            Adds the constant i to the program counter if register j is equal to
            zero.
  0x5iij - BRANCH IF NOT EQUAL TO ZERO.
            Adds the constant i to the program counter if register j is not
            equal to zero.
